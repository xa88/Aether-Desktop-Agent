## ADA 是什么？

**Aether Desktop Agent** 是一款直接运行在 Windows 桌面上的独立 AI 自主智能体。与基于云端的 Copilot 不同，ADA 在本地工作、协调多个 AI 模型，并能控制您的整个操作系统——从文件操作、代码编辑到浏览器自动化和视觉屏幕控制。

ADA 将高性能的 **Rust 核心**与精美的 **React + Electron 界面**相结合，为开发者、高级用户和 AI 爱好者提供无缝体验。

---

## ✨ 核心功能

### 🧠 智能规划与执行
- **双 AI 模型架构** — 独立的"总监"模型负责规划，"工人"模型负责快速执行
- **层级任务规划** — 自动将复杂目标分解为结构化子任务
- **自动错误恢复** — 检测失败并使用替代策略自动重试

### 🖥️ 完整桌面控制
- **屏幕视觉感知** — 捕获并视觉分析您的屏幕以识别 UI 元素
- **鼠标和键盘控制** — 模拟硬件级输入，实现完整的桌面自动化
- **浏览器自动化** — 自主导航网站、填写表单并提取信息

### 💻 集成开发环境
- **内置 Monaco 代码编辑器** — 完整的语法高亮和多语言支持
- **沙箱化代码执行** — 在隔离的执行环境中安全运行脚本
- **集成终端** — 带有流式输出的实时命令执行

### 🤝 多智能体群集
- **智能体编排** — 并行协调多个专业 AI 智能体
- **实时群集 HUD** — 显示所有活跃智能体和任务进度的可视化仪表板
- **总监/工人协议** — 在高能力模型和快速模型之间智能路由

### 🔒 安全与隐私
- **身份库** — 使用 Windows 系统密钥环的企业级密钥管理
- **审计日志** — 每次自主操作的完整时间线，实现完全透明
- **本地优先** — 所有处理均在您的设备上进行；不向外部服务器发送任何数据

### 🧩 可扩展性
- **插件市场** — 兼容 30,000+ VSCode 扩展
- **语音控制** — 麦克风捕获和语音转文字，实现免提操作
- **永久记忆** — 从过去的操作中学习，提升未来性能
- **自我进化** — 分析自身性能并生成改进策略

## 📦 安装

1. 下载最新的 [`ADA-Installer.exe`](https://github.com/xa88/Aether-Desktop-Agent/releases)
2. 运行安装程序并选择安装目录
3. 可选择启用**安装支持库**以自动配置所需环境
4. 从桌面快捷方式启动 **Aether Desktop Agent**

---

## ⚙️ 首次启动设置

首次启动时，打开**设置**进行配置：
- 您的 **LLM 提供商**（OpenAI、Anthropic 或任意 OpenAI 兼容的本地服务器）
- **规划模型** — 用于深度推理任务（例如 GPT-4o）
- **执行模型** — 用于快速响应（例如 GPT-4o-mini、Phi-3）
