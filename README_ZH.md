# 🛰️ Aether Desktop Agent (ADA) - 量子多模态集群

**全球首个分布式、视觉感知、自主进化的操作系统自动化控制器。**

ADA 不仅仅是一个 AI 助手，它是一个由自主代组代理构成的分布式生态系统。它能够感知您的桌面环境，进行 OS 级交互，并跨本地网络扩展以解决极其复杂的任务。

---

## 🚀 愿景：自主操作系统的未来
ADA 弥合了大语言模型与物理操作系统控制之间的鸿沟。通过将高智能的“编排器（Director）”与低成本的“执行器（Worker）”相结合，ADA 同时优化了推理深度与执行速度。

### 👁️ 多模态感知
ADA 可以“看见”您的屏幕。它无需 API 即可识别 UI 元素、按钮和输入框。它通过模拟硬件级的鼠标和键盘事件与您的桌面进行交互。

### 🌐 分布式智能
在您的本地网络中组建 ADA 节点集群。通过我们的安全 P2P 协议，将繁重的计算或特定 OS 任务下发到集群中的其他设备执行。

### 🧩 全球扩展性
集成 **Open VSX 市场**，ADA 可以安装并使用超过 30,000 个 VSCode 兼容插件，无限扩展其工具库。

### 💾 永久记忆
每一次成功的操作都会存储在 ADA 的语义记忆中。它能从自身经验中学习，并找回过去的成功规划，以更快的速度解决新问题。

---

## 🛠️ 核心特性
- **双模型编排**：将规划路由至 GPT-4o，将执行路由至 Phi-3/Llama-3。
- **视觉定点（Visual Grounding）**：自动检测 OS 级自动化的 (x, y) 坐标。
- **Swarm HUD**：实时视觉化跟踪活跃代理及其任务树。
- **Identity Vault**：集成系统钥匙串的企业级机密管理。
- **审计回放**：详尽记录每一个自主动作的时间线，确保完全透明。

---

## 📦 安装 (Windows)
1. 下载最新的 `ADA-Installer.exe`。
2. 运行安装程序并在安装向导中选择“快速安装”。
3. （可选）在安装界面勾选“安装支持库”，以自动配置 Node.js 和 Rust 环境。
